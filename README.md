# Calculadora Java

## 📌 Sobre o projeto
Aplicação desktop de calculadora desenvolvida em Java utilizando Swing (JFrame).  
O projeto tem como objetivo praticar conceitos de lógica de programação, manipulação de eventos e organização de código.

## 🚀 Funcionalidades
- Soma
- Subtração
- Multiplicação
- Divisão
- Porcentagem
- Limpar valores

## 🛠️ Tecnologias utilizadas
- Java
- Swing (JFrame)
- NetBeans

## 🧠 Conceitos aplicados
- Programação orientada a objetos (POO)
- Separação entre Model e View
- Tratamento de eventos (ActionListener)
- Manipulação de interface gráfica

## ▶️ Como executar
1. Clone o repositório
2. Abra no NetBeans
3. Execute o projeto
